# SignalHound Website

Official website for SignalHound - Find Your Lost Bluetooth Devices

## Live Site

This site is hosted on GitHub Pages at: `https://signalhoundapp.github.io/signalhoundapp-website/`

## Pages

- **index.html** - Main landing page with app features and description
- **privacy-policy.html** - Privacy policy (required for App Store submission)

## Local Development

To view locally:
```bash
# Option 1: Python
python3 -m http.server 8000

# Option 2: Node.js
npx http-server

# Then open: http://localhost:8000
```

## Deployment

This site is automatically deployed via GitHub Pages when changes are pushed to the main branch.

To enable GitHub Pages:
1. Go to repository Settings → Pages
2. Under "Source", select "main" branch
3. Click "Save"
4. Site will be live at: https://signalhoundapp.github.io/signalhoundapp-website/

## Privacy Policy URL

For App Store submission, use:
```
https://signalhoundapp.github.io/signalhoundapp-website/privacy-policy.html
```
